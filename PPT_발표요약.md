## 🌟 발표 시 강조할 점

### 1. 랜덤 추천 알고리즘 (가장 중요!)
```
"같은 Pink 색상을 3번 클릭해보겠습니다"
→ 1차: [딸기케이크, 밀크티, 복숭아케이크]
→ 2차: [핑크도넛, 딸기요거트스무디, 밀크티]
→ 3차: [스트로베리필드, 딸기케이크, 딸기블루베리블렌디드]

"매번 다른 조합! 넷플릭스처럼 볼 때마다 새로운 발견!"
```

### 2. 탭 슬라이드 애니메이션
```
"페이지 로드 후 자동으로 슬라이드가 전환됩니다"
→ Coffee (3초) → Donut (3초) → Tea (3초) → 순환

"탭을 클릭하면 즉시 전환되며, 방향도 자동으로 결정됩니다"
→ Coffee → Tea 클릭: 왼쪽 슬라이드
→ Coffee → Donut 클릭: 오른쪽 슬라이드
```

### 3. 색상 기반 추천
```
"색상 심리학을 활용한 감성적 선택"
- Pink: 설렘과 따뜻함
- Navy: 깊이 있는 집중
- Yellow: 밝고 긍정적
```

### 4. 기술적 깊이
```
- Fisher-Yates: O(n) 시간 복잡도
- Redux Toolkit: 전역 상태 관리
- CSS Transform: 60fps 애니메이션
```

## 💡 데모 시연 순서

### 1단계: 색상 선택 (30초)
1. 메인 페이지 → Pink 클릭
2. 모달 팝업 → 감성 메시지 + 랜덤 추천 3개
3. 모달 닫기

### 2단계: 랜덤 추천 (30초)
1. Pink 다시 클릭 → 다른 메뉴 3개
2. Pink 또 클릭 → 또 다른 메뉴 3개
3. "매번 다른 추천!" 강조

### 3단계: 상세 페이지 (30초)
1. 추천 메뉴 중 하나 클릭
2. 상세 페이지 이동
3. **자동 슬라이드 동작** (Coffee → Donut → Tea)
4. 탭 클릭으로 수동 전환

### 4단계: 장바구니 (30초)
1. HOT/ICE 선택
2. 장바구니 담기
3. 장바구니 페이지 → 실시간 계산
4. 수량 조절 → 가격 자동 업데이트

**총 데모 시간: 2분**

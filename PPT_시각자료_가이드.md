## 🎬 데모 영상 제작 가이드

### 1. 메인 플로우 데모 (30초)
```
1. 메인 페이지 진입 (2초)
2. Pink 색상 클릭 (1초)
3. 모달 팝업 + 추천 메뉴 3개 (3초)
4. 모달 닫기 (1초)
5. Pink 다시 클릭 (1초)
6. 다른 메뉴 3개 추천 (3초) ← 랜덤 강조!
7. 추천 메뉴 중 하나 클릭 (1초)
8. 상세 페이지 이동 (2초)
9. 장바구니 담기 (2초)
10. 장바구니 페이지 이동 (2초)
```

### 2. 랜덤 추천 데모 (15초)
```
1. Pink 클릭 → 메뉴 A, B, C (3초)
2. 닫기 (1초)
3. Pink 다시 클릭 → 메뉴 D, E, F (3초)
4. 닫기 (1초)
5. Pink 다시 클릭 → 메뉴 G, H, I (3초)
6. "매번 다른 추천!" 텍스트 오버레이 (2초)
```

### 3. 장바구니 데모 (20초)
```
1. 장바구니에 상품 3개 (2초)
2. 수량 증가 (+) (2초)
3. 가격 자동 계산 (2초)
4. 수량 감소 (-) (2초)
5. 가격 자동 계산 (2초)
6. 상품 삭제 (2초)
7. 총 가격 재계산 (2초)
8. 이름순 정렬 (2초)
```
